# IML_project
Emődi Marcell and Kolláth Gergely's solution to BME's "Introduction to machine learning"/BMETE15AP61, course final project.

Our project aims to create a machine learning model, using KNN, which categorizes music snippets into genres. The model is trained on the GTZAN genre classification dataset, which is the most recommended dataset for the music genre classification project. The dataset contains data in the .wav format.
We plan on crating a console application, so the output is a console string.
